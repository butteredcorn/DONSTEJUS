<h1>Welcome to a game by DONSTEJUS</h1>

### To Run

Open a terminal at the game directory then run:
```bash
npm install
```
To start the game server then run:
```bash
node app.js
```
Open a browser and visit:
```url
http://localhost:3000/
```
Create an account and log in to start the game.
